# 这是一个小说下载器

## 功能介绍：

- 能够将网页中的小说爬取下载
- 能够从各种小说网站搜索小说(未完成)

## 使用建议：
- 建议使用代理，以免被小说网站锁IP

## 已支持的网址：

### [ethxs.com(以太小说网)](http://ethxs.com)

- 此网站的稳定性不高，有时会挂
- 小说存在加密，解码需求较长时间
- 不支持小说内的图片

## 目前已知问题：
